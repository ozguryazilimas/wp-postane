== Description ==

This plugin displays a list of multimedia files in a fancy way. Allows addition for logged in users, and modification for admin users.


== Installation ==

For WordPress 3.+:

1. Download the latest version of the plugin from its page.
1. Upload the .zip through the Plugins/Add new/Upload of the WordPress administrator.
1. Activate the plugin through the Plugins menu of the WordPress administrator.

This plug in will work as is. Just go to '/ugurcum/' in your Wordpress site, like 'http://example.com/ugurcum/'.
