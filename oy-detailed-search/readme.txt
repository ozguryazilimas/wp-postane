== Installation ==

1. Upload the .zip through the Plugins/Add new/Upload of the WordPress administrator.
2. Activate the plugin through the Plugins menu of the WordPress administrator.

This plug in will work as is. Just go to '/ayrintili-ara/' in your Wordpress site, like 'http://example.com/ayrintili-ara/'.
